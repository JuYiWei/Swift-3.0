简介

Strings and Characters 官方文档

  • 字符串String，字符Character，统一使用""即可。
  • 兼容Unicode，使用时会有一点不同。
  • 字符串拼接，支持+运算符，注意不能-。
  • 格式化字符串也不同，Swift使用字符串插值法，不过大体差不多。
  • Swift的String类型与NSString类无缝桥接。

有关String与Foundation和Cocoa，参考Using Swift with Cocoa and Objective-C (Swift 3)

字符，字符串初始化

字符串字面量

let someString = "Some string literal value"

初始化一个空字符串

  • 初始化空字符串，空即为""。
var emptyString = ""               // 空字面量 初始化
var anotherEmptyString = String()  // 初始化方法 初始化
  • 判断字符串是否为空。
if emptyString.isEmpty {
    print("Nothing to see here")
}

字符串可变性

  • 字符串的可变性，Swift统一由常量let与变量var控制。不再区分为NSString与NSMutableString。
let constantString = "Highlander"

var variableString = "Horse"
variableString += " and carriage"

字符串是值类型

  • 值类型，简单的说就是传值时，传的是新拷贝的值，也就是默认copy属性。

字符串字符集

  • characters属性，可以获取字符串本身的所有字符数组。
for character in "Dog!🐶".characters {
    print(character)
}
  • 创建字符，字符也可以转换成字符串。
let exclamationMark: Character = "!"

let catCharacters: [Character] = ["C", "a", "t", "!", "🐱"]
let catString = String(catCharacters)
print(catString)

字符串基本操作

连接字符串和字符

  • 使用+按顺序拼接字符串。
let string1 = "hello"
let string2 = " there"
var welcome = string1 + string2
  • 在现有字符串值的基础上拼接+=。
var instruction = "look over"
instruction += string2
  • 字符串类型提供append()方法，拼接字符串。
let exclamationMark: Character = "!"
welcome.append(exclamationMark)

字符串插值

  • 也就是曾经的格式化字符串，使用\(some)来插入，可以插入各种类型，各种表达式。注意，括号内不能包含反斜杠\，回车，换行。
let multiplier = 3
let message = "\(multiplier) times 2.5 is \(Double(multiplier) * 2.5)"

Unicode

  • 转义特殊字符
\0（空字符） 
\\（反斜杠） 
\t（水平制表） 
\n（换行） 
\r（回车） 
\"（双引号）
\'（单引号）
  • 任意Unicode标量，写成\u{}，其中值为1-8位十六进制数。
let dollarSign = "\u{24}"        // $,  Unicode scalar U+0024
let blackHeart = "\u{2665}"      // ♥,  Unicode scalar U+2665
let sparklingHeart = "\u{1F496}" // 💖, Unicode scalar U+1F496
  • 由一个或多个的Unicode标量，结合组成单一可读字符。
let eAcute: Character = "\u{E9}"                         // é
let combinedEAcute: Character = "\u{65}\u{301}"          // e +  ́ = é 
// 上面两个是相同的字符。
  • 再比如。
let precomposed: Character = "\u{D55C}"                  // 한
let decomposed: Character = "\u{1112}\u{1161}\u{11AB}"   // ᄒ, ᅡ, ᆫ ==> 한
  • 三比如。
let enclosedEAcute: Character = "\u{E9}\u{20DD}"
// enclosedEAcute is é⃝
  • 四比如。
let regionalIndicatorForUS: Character = "\u{1F1FA}\u{1F1F8}"
// regionalIndicatorForUS is 🇺🇸

字符计数

  • 获取字符串的字符集，也就是字符数组，那么就可以获取字数了。注意，由于字符是可以由Unicode标量组合成而成的，所以组合的字符计数为1。
var word = "cafe"
// cafe == 4
 
word += "\u{301}"
// café == 4

// 拼接了 \u{301} 相当于组合标量，把最后的e，加了一个声调，但视为一个字符。

访问和修改字符串字符集

  • 使用下标语法。
  • 使用其方法和属性。

字符串索引

  • 位置属性，startIndex和endIndex，获取字符串字符的起始和结束位置。注意，是索引类型而不是整形。
var startIndex = name.startIndex
var endIndex = name.endIndex
  • 索引类型位置偏移，注意结合字符串，避免越界。
var afterIndex = name.index(after: startIndex)
var beforeIndex = name.index(before: endIndex)
var offsetIndex = name.index(startIndex, offsetBy: 3)
  • 下标语法，获取指定位置的字符。
var name = "Li Lei"
var ch = name[startIndex];

插入和删除

  • 插入字符insert(_:at:)，插入字符串insert(contentsOf:at:)。注意，插入字符串，即字符集合。
var name = "Li Lei"
name.insert("!", at: name.endIndex)
name.insert(contentsOf: "hello ".characters, at: name.startIndex)
  • 删除字符remove(at:)，删除字符串removeSubrange(_:)。
var welcome = "hello Li Lei !"
welcome.remove(at: welcome.index(before: welcome.endIndex))
let range = welcome.startIndex..<welcome.index(welcome.startIndex, offsetBy: 5)
welcome.removeSubrange(range)
print(welcome)

字符串比较

  • 直接比较。
  • 前缀比较。
  • 后缀比较。

直接比较

  • 比较使用==和!=.字符串比较时，逐个字符单一比较。
let name = "Li Lei"
let name2 = "LiLei"

if name != name2 {
    print("not equal")
}
  • Unicode标量组合成的字符与单一标量的字符可能视为一致。 
let cafe1 = "caf\u{E9}"
let cafe2 = "caf\u{65}\u{301}"

if cafe1 == cafe2 {
    print("café")
}
  • 不同单一标量的字符，视为不一致，尽管看起来一样。
let latinCapitalLetterA: Character = "\u{41}"
let cyrillicCapitalLetterA: Character = "\u{0410}"
 
if latinCapitalLetterA != cyrillicCapitalLetterA {
    print("not equal")
}

前缀比较

  • hasPrefix(_:)。
let info = "Li Lei，male"
if info.hasPrefix("Li") {
    print("我是Li家的")
}

后缀比较

  • hasSuffix(_:)。
let info = "Li Lei，male"
if info.hasPrefix("male") {
    print("我是男的")
}

字符串编码

准备字符串，下面开始操作。

let dogString = "Dog‼🐶"

UTF-8表示

  • utf8属性访问String的UTF-8值。此属性是String.UTF8View类型，也就是UInt8类型。
for codeUnit in dogString.utf8 {
    print("\(codeUnit) ", terminator: "")
}
// 68 111 103 226 128 188 240 159 144 182
  • [0，3)位十进制codeUnit值（68，111，103）代表D，o，g。
  • [3，6)位十进制codeUnit值（226，128，188）代表!!。
  • [6,10)位十进制codeUnit值（240，159，144，182）代表🐶。

UTF-16表示

  • utf16属性访问String的UTF-16值。此属性是String.UTF16View类型，也就是UInt16类型。
for codeUnit in dogString.utf16 {
    print("\(codeUnit) ", terminator: "")
}
// Prints "68 111 103 8252 55357 56374 "
  • [0，3)位十进制codeUnit值（68，111，103）代表D，o,g）。
  • [3，4)位十进制codeUnit值（8252）代表!!。
  • [4，6)位十进制codeUnit值（55357和56374）代表🐶。

Unicode标表示

  • unicodeScalars属性访问String的Unicode值。此属性是UnicodeScalarView类型，也就是UnicodeScalar类型。每UnicodeScalar有一个value返回21位标量，也就是UInt32类型。
for scalar in dogString.unicodeScalars {
    print("\(scalar.value) ", terminator: "")
}
// Prints "68 111 103 8252 128054 "
  • [0，3)位UnicodeScalar（68，111，103）代表D，o，g。
  • [3，4)位十进制codeUnit值（8252）代表!!。

  • [4，5)位十进制codeUnit值（128054）代表🐶。

  • 作为替代查询他们的value属性，每一个UnicodeScalar值也可以被用来构造一个新String值。

for scalar in dogString.unicodeScalars {
    print("\(scalar) ")
}
// D
// o
// g
// ‼
// 🐶
