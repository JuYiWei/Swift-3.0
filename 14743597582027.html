Functions 官方文档

函数的定义和调用

  • 基础函数定义。func前缀表示函数，函数名称（参数1，参数2） -> 返回类型。
func greet(person: String) -> String {
    let greeting = "Hello, " + person + "!"
    return greeting
}
  • 函数调用
print(greet(person: "Anna"))
print(greet(person: "Brian"))

函数的参数和返回值

无参数

  • 没有参数，但是包含参数的()，不能省略。
func sayHelloWorld() -> String {
    return "hello, world"
}
print(sayHelloWorld())

多参数

  • 多参数，逗号分隔。
func greet(person: String, alreadyGreeted: Bool) -> String {
    if alreadyGreeted {
        return greetAgain(person: person)
    } else {
        return greet(person: person)
    }
}
print(greet(person: "Tim", alreadyGreeted: true))

无返回

  • 未定义返回的函数，实际返回void的特殊值，即()。
func greet(person: String) {
    print("Hello, \(person)!")
}
greet(person: "Dave")
  • 函数的返回值可以在调用时忽略。
func printAndCount(string: String) -> Int {
    print(string)
    return string.characters.count
}
func printWithoutCounting(string: String) {
    let _ = printAndCount(string: string)
}
printAndCount(string: "hello, world")

多返回

  • 使用一个元组类型的返回类型，使函数返回多个值。
func minMax(array: [Int]) -> (min: Int, max: Int) {
    var currentMin = array[0]
    var currentMax = array[0]
    for value in array[1..<array.count] {
        if value < currentMin {
            currentMin = value
        } else if value > currentMax {
            currentMax = value
        }
    }
    return (currentMin, currentMax)
}

返回可选类型

func minMax(array: [Int]) -> (min: Int, max: Int)? {
    if array.isEmpty { return nil }
    var currentMin = array[0]
    var currentMax = array[0]
    for value in array[1..<array.count] {
        if value < currentMin {
            currentMin = value
        } else if value > currentMax {
            currentMax = value
        }
    }
    return (currentMin, currentMax)
}
  • 使用可选绑定来，解析可选类型返回值的内容。
if let bounds = minMax(array: [8, -6, 2, 109, 3, 71]) {
    print("min is \(bounds.min) and max is \(bounds.max)")
}

指定参数标签

  • 标签用于外部提示，默认与参数名称一致。
func totalName(firstName a:String, lastName b:String) -> String {
    return a + b
}
totalName(firstName: "Li", lastName: "Lei")

省略参数标签

func someFunction(_ firstParameterName: Int, secondParameterName: Int) {
    // In the function body, firstParameterName and secondParameterName
    // refer to the argument values for the first and second parameters.
}
someFunction(1, secondParameterName: 2)

默认参数值

func cal(a : Int, b : Int = 18) -> Int{
    return a+b
}

cal(a: 12)
cal(a: 12, b: 12)

可变参数的参数

  • 就是把参数变成了数组，且最多只能有一个可变参数。
func sumof(_ numbers: Int...) -> Int {
    var sum = 0 
    for a in numbers {
        sum += a
    }
    return sum
}

sumof(1,2,3,4,5)

inout参数

  • 直接修改内存地址数据。
var a = 111
var b = 222

func mySwap(a: inout Int, b: inout Int) {
    let temporaryA = a
    a = b
    b = temporaryA
}

mySwap(a: &a, b: &b)

函数类型

  • 2个Int类型的参数，并返回1个Int类型的数据。该函数类型为(Int, Int) -> Int。
func addTwoInts(_ a: Int, _ b: Int) -> Int {
    return a + b
}

使用函数类型

  • 根据函数类型，可以定义函数变量。
func addTwo(a: Int, b: Int) -> Int {
    return a + b
}

var myfunc: (Int, Int) -> Int = addTwo;

myfunc(2, 3)

函数类型作为参数

// 函数
func addTwo(a: Int, b: Int) -> Int {
    return a + b
}

// 函数类型做参数
func myFunc(add:(Int, Int)-> Int) -> Int {
    return add(2, 3) + 10
}

// 传入函数
myFunc(add: addTwo)

返回函数类型

func addTwo(a: Int, b: Int) -> Int {
    return a + b
}

func myFunc() -> (Int, Int)-> Int {
    return addTwo;
}

myFunc()(1,2)
