声明

在斯威夫特，有三种语句：简单语句，编译器的控制语句和控制流语句。简单的语句是最常见的，并且由是表达式或声明。编译器的控制语句允许程序改变编译器的行为的方面，并包括一个条件编译块和线路控制语句。

控制流语句用于控制执行中的程序的流程。有斯威夫特几种类型的控制流语句，包括循环语句，分支语句，并控制转移语句。循环语句允许重复执行的代码块，分支语句允许在满足一定的条件只执行的代码一定的块，并且控制转移语句提供一种方式来改变在其中执行代码的顺序。此外，雨燕提供了一个do声明，介绍范围，捕捉和处理错误，而defer只是在当前范围退出之前运行的清除操作语句。

分号（;）可以有选择地出现在任何语句后，用来当他们出现在同一行分隔多个语句。

一个switch语句也可以包括一个默认的情况下，通过引入default关键字。只有在没有其他情况下，控制表达式匹配默认情况下中的代码被执行。一个switch声明只能包含一个默认情况下，它必须出现在年底switch的语句。

虽然模式匹配操作的实际执行顺序，并且特别在例模式的评价顺序，是不确定的，在一个模式匹配switch语句的行为如同在源进行评价顺序，也就是说，该命令在它们出现在源代码。其结果是，如果有多个的情况下包含评估为相同的值，因而可以匹配控制表达式的值模式，程序执行只在源顺序第一匹配壳体内的代码。

开关语句必须面面俱到

在夫特，控制表达式的类型的每个可能的值必须的情况中的至少一种模式的值相匹配。当这根本是不可行的（例如，当控制表达式的类型就是Int），可以包含一个默认的情况下，以满足要求。

执行不会通过案例隐秋

如果多个defer语句出现在同一范围内，它们出现的顺序就是他们执行的顺序相反。执行最后一次defer在给定的范围内声明的第一意味着最后声明里defer声明可以引用将由其他清理资源defer语句。

func f() {
defer { print(First) }
defer { print(Second) }
defer { print(Third) }
}
f()
// Prints Third
// Prints Second
// Prints First
在该声明defer语句不能在境外传递程序控制defer声明。
