简介

Collection Type 官方文档

  • Set、Array、Dictionary，本身较为常规。
  • 常量let和变量var，控制数据是否可变。

关于泛型新增项目，详情看泛型章节。

数组 Array

  • 有序存储相同类型的值。
  • Swift的Array类型无缝桥接Foundation的NSArray类。

有关使用的详细信息Array与Foundation和Cocoa，参考Using Swift with Cocoa and Objective-C (Swift 3)

数组初始化

  • 初始化方式Array<Element>，其中Element为Array存储值的类型。
  • 简写[Element]，推荐简写方式。

创建空数组

  • 初始化方法和简写初始化空数组。已知数据类型时，可以直接使用[]，简写创建数组。
var array = Array<Int>()
var array2: Array<Int> = []

var arr = [Int]()
var arr2: [Int] = []

创建带有默认值的Array

  • 默认值和默认数量的初始化。
var threeDoubles = Array(repeating: 0.0, count: 3)

字面量初始化数组

  • 常规，与正常初始化没差啊？！有差的是，Int也可以加入数组了，而不是OC那样需要对象，可见Swift都是对象？Int也是对象？
var shoppingList: [String] = ["Eggs", "Milk"]
  • 类型推断，简写
var shoppingList = ["Eggs", "Milk"]

访问和修改数组

  • count属性，查询数组个数。
var count = arr1.count;
  • isEmpty属性，检查count是否为0。
if arr.isEmpty {
    print("数组空哒")
}
  • append拼接。
arr.append("aaa");
arr.append(contentsOf: ["a","b"])
arr += ["zzz"]
  • 下表语法，获取并修改指定内容。
var firstItem = arr[0]
arr[0] = "Six eggs"
arr[4...6] = ["Bananas", "Apples"]
  • insert插入。
arr.insert("qq", at: 0)
arr.insert(contentsOf: ["q","w","e"], at: 0)
  • remove删除并返回被删除元素。
arr.remove(at: 0)

遍历数组

  • 直接遍历数组。
var arr = ["a","b","c","d","e","f"]
for item in arr {
    print(item)
}
  • 枚举遍历，对数组enumerated()之后遍历，可以获取(index，value)的元组。
var arr = ["a","b","c","d","e","f"]
for (index, value) in arr.enumerated() {
    print(index, value)
}

集合 Set

  • 集合没有排序，不会重复。
  • Swift的Set类型无缝桥接Foundation的NSSet类。

有关使用的详细信息Set与Foundation和Cocoa，参考Using Swift with Cocoa and Objective-C (Swift 3)

集合元素 hash

  • 集合中的数据类型必须可以hash，当然String，Int，Double，Bool，以及枚举都默认支持。也可以自定义类型，支持Hashable协议即可。

有关协议的详细信息，请参阅协议。

集合初始化

  • Set，其中Element是允许的类型。与数组不同，Set不具备等效的简写形式。

创建空集合

  • 初始化方法初始化空数组。已知数据类型时，可以直接使用[]，简写创建集合。
var s2 = Set<Int>()
var s:Set<Int> = []

字面量初始化集合

  • 初始化集合与数组不同，必须指定类型为Set。
var favoriteGenres: Set<String> = ["Rock", "Classical", "Hip hop"]
  • 类型推断，可以简写。
var favoriteGenres: Set = ["Rock", "Classical", "Hip hop"]

访问和修改集合

  • count，isEmpty，insert，remove，方法与数组一致，略。

  • contains，检查是否包含。

var sets: Set = [1,2,3,4]
if sets.contains(1) {
    print("contain")
}

遍历集合

  • 一般遍历。
var sets: Set = [1,2,3,4]
for item in sets {
    print(item)
}
  • 排序遍历。
var sets: Set = [1,2,3,4]
for item in sets.sorted() {
    print(item)
}

集合运算

  • 基本操作，例如交集，并集，包含等，集合数学的集合很好理解。
var set1: Set = [1,2,3,4]
var set2: Set = [3,4,5,6]


// 交集 3，4
var s1 = set1.intersection(set2)


// 并集除交集 1，2，5，6
var s2 = set1.symmetricDifference(set2)


// 交集 1，2，3，4，5，6
var s3 = set1.union(set2)


// A中非B 1，2
var s4 = set1.subtracting(set2)

集合比较

var set1: Set = [1,2]
var set2: Set = [1,2,3]

// 是否一致
var b0 set1 == set2


// 1 是否 被2 包含
var b1 = set1.isSubset(of: set2)


// 1 是否 包含 2
var b2 = set1.isSuperset(of: set2)


// 1 是否是 2 的子集，且不同
var b3 = set1.isStrictSubset(of: set2)
// 1 是否是 2 的超级，且不同
var b4 = set1.isStrictSuperset(of: set2)


// 1 和 2 是否完全不相交
var b5 = set2.isDisjoint(with: set1)

Dictionary

  • Dictionary是键值对，同样没有指定顺序。

- Swift的Dictionary类型无缝桥接Foundation的NSDictionary类。

有关使用的详细信息Dictionary与Foundation和Cocoa，参考Using Swift with Cocoa and Objective-C (Swift 3)

字典初始化

  • Dictionary<Key, Value>。
  • 字典Key必须符合Hashable协议。
  • 字典的简写[Key: Value]。推荐简写。

创建空字典

  • 类似数组。
var dictionary = [Int: String]()
var dictionary2: Dictionary<Int, String> = [:]

var dict = [Int: String]()
var dict2: [Int: String] = [:]

字面量初始化字典

  • 初始化方式差不多。
var dictionary: [String: String] = ["YYZ": "Toronto Pearson", "DUB": "Dublin"]
  • 类型推断，简写。
var dict = ["YYZ": "Toronto Pearson", "DUB": "Dublin"]

访问和修改词典

  • count，isEmpty属性，获取Dictionary数量，判断是否为空（略）。

  • 使用下标语法，新增键值对。

airports["LHR"] = "London"
  • 使用下标语法，修改键值对。
airports["LHR"] = "London Heathrow"
  • 修改键值对，updateValue(_:forKey:)，方法会返回旧值，如果不存在则返回nil，注意是可选类型。
if let oldValue = airports.updateValue("Dublin Airport", forKey: "DUB") {
    print("The old value for DUB was \(oldValue).")
}
  • 使用下标语法检索键值对，获得的也是可选类型。
if let airportName = airports["DUB"] {
    print("The name of the airport is \(airportName).")
} else {
    print("That airport is not in the airports dictionary.")
}
  • 使用下标语法，直接赋值为nil，可删除该键值对。
airports["APL"] = "Apple International"
airports["APL"] = nil
  • removeValue(forKey:)删除方法。如果存在，删除并返回被移除的值，不存在则返回nil。
if let removedValue = airports.removeValue(forKey: "DUB") {
    print("The removed airport's name is \(removedValue).")
} else {
    print("The airports dictionary does not contain a value for DUB.")
}

遍历Dictionary

  • for- in循环遍历键-值对。字典中的每个键值对返回一个(key, value)元组。
for (airportCode, airportName) in airports {
    print("\(airportCode): \(airportName)")
}
  • 访问检索字典的所以键或值，keys和values。
for airportCode in airports.keys {
    print("Airport code: \(airportCode)")
}
 
for airportName in airports.values {
    print("Airport name: \(airportName)")
}
