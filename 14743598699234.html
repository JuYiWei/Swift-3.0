Inheritance 官方文档

  • 继承父类的各种属性与方法。
  • 可以选择性的重写。
  • 也可以强制防止子类重写。

定义一个基类

  • 与OC不同，Swift新建的类不指定继承，那么它就是基类。
class Vehicle {
    var currentSpeed = 0.0
    var description: String {
        return "traveling at \(currentSpeed) miles per hour"
    }
    func makeNoise() {
        // do nothing - an arbitrary vehicle doesn't necessarily make a noise
    }
}

子类化

  • 子类继承现有类，拓展更多功能，新增属性与方法。
class Bicycle: Vehicle {
    var hasBasket = false
}

重写

  • 子类可以提供一个实例方法，类型的方法，实例属性，类型属性，或下标，覆盖原本从父继承的属性或方法。override关键字，系统也会检查是否可以重写。

重载方法

  • 可以完全覆盖父类方法，也可以选择使用super调用父类方法，再添加新实现。
class Train: Vehicle {
    override func makeNoise() {
        super.makeNoise()
        print("Choo Choo")
    }
}
let train = Train()
train.makeNoise()

重写属性getter和setter

  • 具体set和get看属性章节，重写属性与重写方法基本一致。
class Car: Vehicle {
    var gear = 1
    override var description: String {
        return super.description + " in gear \(gear)"
    }
}

let car = Car()
car.currentSpeed = 25.0
car.gear = 3
print("Car: \(car.description)")

重写属性观察

  • 具体看属性章节，重写属性观察与重写方法基本一致。
class AutomaticCar: Car {
    override var currentSpeed: Double {
        didSet {
            gear = Int(currentSpeed / 10.0) + 1
        }
    }
}

let automatic = AutomaticCar()
automatic.currentSpeed = 35.0
print("AutomaticCar: \(automatic.description)")

防止覆盖

  • 使用final，表示子类不能重写。重写会导致报错。
class Vehicle {
    var currentSpeed = 0.0
    var description: String {
        return "traveling at \(currentSpeed) miles per hour"
    }
    
    final func makeNoise() {
         print("super")
    }
}

// error
class Train: Vehicle {
    override func makeNoise() {
        super.makeNoise()
        print("Choo Choo")
    }
}
