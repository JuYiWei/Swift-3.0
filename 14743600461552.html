简介

Generics 官方文档

通用代码，使您能够编写灵活，可重复使用的功能和类型，可以与任何类型的工作，受您定义的要求。你可以写，避免重复，并表示在一个明确的，抽象的方式意图的代码。

泛型是斯威夫特的最强大的功能之一，很多雨燕标准库的建立与通用代码。事实上，你已经使用整个泛型语言指南，即使你没有意识到这一点。例如，斯威夫特Array和Dictionary类型都是泛型集合。您可以创建包含数组Int值，或者保存一个数组String值，或者确实可以在斯威夫特被创建的任何其他类型的数组。同样，您可以创建一个字典存储任何指定类型的值，并有上什么类型可以是没有任何限制。

泛型解决问题

  • 类型的通用化，是函数更加灵活。如果没有泛型，那么交换函数需要非常多的类型，二使用泛型则通用成一个即可。
func swapTwoInts(_ a: inout Int, _ b: inout Int) {
    let temporaryA = a
    a = b
    b = temporaryA
}

func swapTwoStrings(_ a: inout String, _ b: inout String) {
    let temporaryA = a
    a = b
    b = temporaryA
}
 
func swapTwoDoubles(_ a: inout Double, _ b: inout Double) {
    let temporaryA = a
    a = b
    b = temporaryA
}

...

泛型函数

  • 接上面遗留问题，下面是泛型交换的函数。
func swapTwoValues<T>(_ a: inout T, _ b: inout T) {
    let temporaryA = a
    a = b
    b = temporaryA
}

类型参数

  • 也就是上述<T>，中的T，表示函数支持的参数类型，支持多个类型参数，逗号连接。
func testFunc<String, Int>(_ a: inout String, _ b: inout Int) {
    
}

命名类型参数

  • 一般字典中使用时Key，Value，数组中是Element，通常情况下无意义时，使用T，U，V这类字母表明。

泛型类型

类型约束

类型约束语法

  • 指定类型
func someFunction<T: SomeClass, U: SomeProtocol>(someT: T, someU: U) {
    // function body goes here
}

类型约束举例

  • 表示了符合Equatable协议的T类型。而不是任何类型了。使用了协议约束。
func findIndex<T: Equatable>(of valueToFind: T, in array:[T]) -> Int? {
    for (index, value) in array.enumerated() {
        if value == valueToFind {
            return index
        }
    }
    return nil
}

相关类型

当定义一个协议，它有时是有益的声明一个或多个相关联的类型作为该协议的定义的一部分。一个相关联的类型给出的占位符的名称被用作协议的一部分的类型。实际类型用于该相关类型没有指定，直到该协议被采用。相关类型与指定的associatedtype关键字。

协议相关类型

  • 协议中泛型
protocol Container {
   associatedtype ItemType
    
   func append(item: ItemType)
}

class SomeClas: Container {
    typealias ItemType = String
    
    func append(item: ItemType) {
        print(item)
    }
}

扩展添加相关

protocol Container {
    associatedtype ItemType    
    func append(item: ItemType)
}

class SomeClas {
    func log() {
        print("log")
    }
}

extension SomeClas: Container {
    typealias ItemType = String
    func append(item: String) {
        print(item)
    }
}

WHERE 约束

protocol Container {
    associatedtype ItemType   
    func append(item: ItemType)
    var count: Int { get }
}

class A: Container {
    typealias ItemType = String
    func append(item: String) {
        print(item)
    }
    var count: Int = 1
}

func allItemsMatch<C1: Container, C2: Container>
    (_ someContainer: C1, _ anotherContainer: C2) -> Bool
    where C1.ItemType == C2.ItemType, C1.ItemType: Equatable {
        
        if someContainer.count != anotherContainer.count {
            return false
        }
        
        return true
}

var a = A()
a.count = 1

var b = A()
b.count = 2

var ok: Bool = allItemsMatch(a, b)
