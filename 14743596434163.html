简介

Basics Operators 官方文档

  • 运算符是一个特殊的符号，用来检查、更改、合并等。

  • Swift支持大多数标准C运算符，并新增了一些检查功能，以消除常见的编码错误。赋值运算符=没有返回值，以防止误用==。算术运算符（+，-，*，/，%等等）检测并禁止值溢出，以避免意外的结果。可以通过使用Swift自定义溢出处理。

  • 迅速还提供了两个新的范围表达方式（a..<b和a...b），方便快捷。

  • 本章介绍了Swift常用的操作符，以及特有高级操作符。

术语

运算符分为：一元，二元或三元。

  • 一元运算符：一个目标进行操作，如-a（负a）。
  • 二元运算符：两个目标进行操作，如2 + 3。
  • 三元运算符：三个目标进行操作，C，Swift只有一个三元运算符（a ? b : c）。

赋值运算符

  • 赋值运算符a = b，用于初始化或更新值。
let b = 10
var a = 5
a = b
  • 如果赋值的是元组，前后对多个变量或者常量。
let (x, y) = (1, 2)
  • 不像在C和Objective-C中的赋值操作符，Swift赋值操作符本身不返回值。可以防止=误用==二产生错误。
if x = y {
    // This is not valid, because x = y does not return a value.
}

算术运算符

四则运算符

Swift所有数字类型支持四个标准的算术运算符。

加​（+）
减（-）
乘（*）
除（/）

  • 不像C和Objective-C的算术运算符，Swift算术运算符不允许值溢出。可以通过使用Swift的溢出运算符处理溢出。（详情看溢出运算）

  • 加法运算符还支持String类型。

"hello, " + "world"  // equals "hello, world"

求余运算符

  • 求余数运算符a % b，工作原理a = b * x + n，n即为余数。特殊的是，Swift中，负数也支持求余。
9 % 4    // 9 = 4 * 2 + 1
-9 % 4   // -9 = 4 * -2 + -1
           // -9 == 4 * -3 + 3
负数求余时，这里选择了第一种方式！

一元减运算符、一元加运算符（负正）

  • 数值可以使用前缀-，为一元减运算符。类似的有+，为一元加运算符。
let three = 3
let minusThree = -three       // minusThree equals -3
let plusThree = -minusThree   // plusThree equals 3, or "minus minus three"

let minusSix = -6
let alsoMinusSix = +minusSix  // alsoMinusSix equals -6
  • 虽然一元加运算实际上并没有做任何事情，可以用它与元减运算符作为对称代码。

复合赋值运算符

  • 像C，Swift提供复合赋值运算符，例如加法赋值运算符+=。
var a = 1
a += 2
  • 复合赋值运算符不返回值。例如，你可以不写let b = a += 2。
  • 不支持 ++，--。

比较运算符

  • Swift支持所有标准的C比较运算符。
等于（a == b）
不等于（a != b）
大于（a > b）
小于（a < b）
大于或等于（a >= b）
小于或等于（a <= b）

  • Swift还提供了两个类型比较运算符===和!==，用来测试两个对象引用是否指向同一个对象实例。欲了解更多信息，请参见类和结构。

  • 每个比较操作符返回一个Bool值来表示语句是否为真。

1 == 1   // true because 1 is equal to 1
2 != 1   // true because 2 is not equal to 1
2 > 1    // true because 2 is greater than 1
1 < 2    // true because 1 is less than 2
1 >= 1   // true because 1 is greater than or equal to 1
2 <= 1   // false because 2 is not less than or equal to 1
  • 比较运算符经常使用的条件语句，如if语句。
let name = "world"
if name == "world" {
    print("hello, world")
} else {
    print("I'm sorry \(name), but I don't recognize you")
}

  • 欲知更多有关if声明，请参阅控制流。

  • 还可以比较具有相同数量的值的元组，需要确保对应的每个元组中的值可比。例如，Int和String可以比较，Bool无法比较。

  • 元组进行比较，从左至右，一个一个对应的比较，直至两个值不相等。如果所有的元素都相等，则元组相等的。例如：

(1, "zebra") < (2, "apple")   // true because 1 is less than 2
(3, "apple") < (3, "bird")    // true because 3 is equal to 3, and "apple" is less than "bird"
(4, "dog") == (4, "dog")      // true because 4 is equal to 4, and "dog" is equal to "dog"
  • Swift标准库，表示元组比较需要少于七个元素。

三元条件运算符

  • question?answer1:answer2 比较常规。作用如下：
if question {
    answer1
} else {
    answer2
}
  • 举例。
let contentHeight = 40
let hasHeader = true
let rowHeight = contentHeight + (hasHeader ? 50 : 20)
  • 上面例子的原版。
let contentHeight = 40
let hasHeader = true
let rowHeight: Int
if hasHeader {
    rowHeight = contentHeight + 50
} else {
    rowHeight = contentHeight + 20
}
  • 合理使用三元运算符，避免过于复杂。

nil合并运算符

  • 强制解析可选类型的简写版本，a ?? b，如果a包含一个值，那么解析a的值，忽略b的值，反之返回默认b的值。下面是同理的三元运算。
a != nil ? a! : b
  • 举个例子，可选类型a为nil时
let defaultColorName = "red"
var userDefinedColorName: String?   // defaults to nil 
var colorNameToUse = userDefinedColorName ?? defaultColorName
  • 可选类型，不为nil时
userDefinedColorName = "green"
colorNameToUse = userDefinedColorName ?? defaultColorName

范围运算符

  • Swift包括两个范围运算符a...b和a..<b。

  • 封闭范围操作符a...b，定义范围[a,b]，包括a和b，且a不大于b。经常使用在for-in中

for index in 1...5 {
    print("\(index) times 5 is \(index * 5)")
}
  • 半封闭范围操作符a..<b，定义范围[a,b)，不包括b，a不大于b。经常使用在数组索引中
let names = ["Anna", "Alex", "Brian", "Jack"]
let count = names.count
for i in 0..<count {
    print("Person \(i + 1) is called \(names[i])")
}

逻辑运算符

  • 逻辑运算符修改或合并布尔值true和false。Swift支持基于C语言中的三个标准逻辑运算符：
逻辑NOT（ !a）
逻辑与（a && b）
逻辑或（a || b）
  • 逻辑NOT运算符!a，反转一个布尔值，true变false和false变true。
let allowedEntry = false
if !allowedEntry {
    print("ACCESS DENIED")
}
  • 逻辑AND运算符a && b，两个值都为true，则整体为true，反之false。如果第一值false，第二值不会判断。
let enteredDoorCode = true
let passedRetinaScan = false
if enteredDoorCode && passedRetinaScan {
    print("Welcome!")
} else {
    print("ACCESS DENIED")
}
  • 逻辑OR运算符a || b，两个值只要有一个为true，则整体为true，反之false。如果第一个为true，第二个同样不会判断。
let hasDoorKey = false
let knowsOverridePassword = true
if hasDoorKey || knowsOverridePassword {
    print("Welcome!")
} else {
    print("ACCESS DENIED")
}
  • 结合逻辑运算符，可以将多个逻辑运算符来创建更长的复合表达式。
if enteredDoorCode && passedRetinaScan || hasDoorKey || knowsOverridePassword {
    print("Welcome!")
} else {
    print("ACCESS DENIED")
}
  • Swift的逻辑运算符&&和||是左关联的，具有多个逻辑运算符复合表达式先评估最左边的子表达式。添加括号，使一个复杂的表达式更容易阅读。
if (enteredDoorCode && passedRetinaScan) || hasDoorKey || knowsOverridePassword {
    print("Welcome!")
} else {
    print("ACCESS DENIED")
}
