简介

Methods 官方文档

  • 方法是与特定类型的相关联的函数。
  • 类，结构和枚举都可以定义实例方法。
  • 类，结构和枚举也可以定义类型方法。
  • 区别OC的是结构和枚举也支持方法。

实例方法

  • 实例方法，就是函数嘛。
class Counter {
    var count = 0
    func increment() {
        count += 1
    }
    func increment(by amount: Int) {
        count += amount
    }
    func reset() {
        count = 0
    }
}

self属性

  • 实例拥有self的隐藏属性，即实例本身。如果没有歧义，也可以省略self。
func increment() {
    self.count += 1
}

实例方法修改值类型属性

  • 结构和枚举是值类型。默认情况下，值类型的属性，不能从它的实例方法中修改。mutating关键字描述函数，隐性添加了self，使其能修改实例属性。注意常量的结构或者枚举，是不能修改的。
struct Point {
    var x = 0.0, y = 0.0
    mutating func moveBy(x deltaX: Double, y deltaY: Double) {
        x += deltaX
        y += deltaY
    }
}
var somePoint = Point(x: 1.0, y: 1.0)
somePoint.moveBy(x: 2.0, y: 3.0)
print("The point is now at (\(somePoint.x), \(somePoint.y))")

实例方法修改实例自身

  • mutating使其隐含self属性，所以也可以直接修改自身。
struct Point {
    var x = 0.0, y = 0.0
    mutating func moveBy(x deltaX: Double, y deltaY: Double) {
        self = Point(x: x + deltaX, y: y + deltaY)
    }
}
  • 枚举同样适用。
enum TriStateSwitch {
    case off, low, high
    mutating func next() {
        switch self {
        case .off:
            self = .low
        case .low:
            self = .high
        case .high:
            self = .off
        }
    }
}
var ovenLight = TriStateSwitch.low
ovenLight.next()
ovenLight.next()

类型方法

  • 类型方法内部的隐性self指向类本身，也就可以使用类型属性。类型方法可以使用static关键词，类也可以使用class关键词。支持类，结构，枚举。
class SomeClass {
    class func someTypeMethod() {
        // type method implementation goes here
    }
}
SomeClass.someTypeMethod()
