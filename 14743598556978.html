下标

类，结构和枚举可以定义标，这是用于访问的集合，列表或序列的部件元素的快捷方式。您可以使用标来设置和读取索引值，而无需进行设置和检索单独的方法。例如，您可以访问在元素Array实例作为someArray[index]和元素的Dictionary实例作为someDictionary[key]。

您可以定义多个标为单一类型，及相应的标超载使用基于传递给下标索引值的类型来选择。标不局限于单一的维度，您可以定义多个输入参数标来满足您的自定义类型的需求。

标语法

标允许您通过实例名称后写在方括号中的一个或多个值来查询一个类型的实例。他们的语法类似于两个实例方法语法和计算的属性语法。编写与标定义subscript关键字，并指定一个或多个输入参数和返回类型，以同样的方式作为实例方法。不像实例方法，下标可以读写或只读。这种行为是由相同的方法计算性能的getter和setter沟通：

subscript(index: Int) -> Int {
get {
// return an appropriate subscript value here
}
set(newValue) {
// perform a suitable setting action here
}
}
的类型newValue是相同的标的返回值。与计算性能，你可以选择不指定setter方法的(newValue)参数。所谓默认的参数newValue提供给您的二传手，如果你不自己提供一个。

作为具有只读属性计算，你可以删除get的只读标关键字：

subscript(index: Int) -> Int {
// return an appropriate subscript value here
}
这里的一个只读标实现，它定义了的一个例子TimesTable的结构来表示Ñ整数-times表：

struct TimesTable {
let multiplier: Int
subscript(index: Int) -> Int {
return multiplier * index
}
}
let threeTimesTable = TimesTable(multiplier: 3)
print(six times three is (threeTimesTable[6]))
// Prints six times three is 18
在这个例子中，一个新的实例TimesTable被创建以代表三次的表。这是通过一个值所指示3的结构是initializer作为值要用于实例的multiplier参数。

可以查询threeTimesTable通过调用其下标，如图调用实例threeTimesTable[6]。这就要求在三次表，它返回的值第六项18，或3时间6。

注意

一个Ñ -times表是基于一个固定的数学规则。这是不恰当设置threeTimesTable[someIndex]到一个新的值，并且因此对于下标TimesTable被定义为只读标。

标用法

“标”的确切含义取决于使用它的上下文。下标通常用作用于访问在一个集合，列表或序列的部件元素的快捷方式。你可以自由的实现您的特定类或结构的功能最合适的标的方式。

例如，迅捷的Dictionary类型实现一个下标设置和检索存储在该值Dictionary的实例。您可以通过提供标括号内的字典中的主要类型的密钥，并指定字典的值类型为下标值设定在字典中的值：

var numberOfLegs = [spider: 8, ant: 6, cat: 4]
numberOfLegs[bird] = 2
上面的例子定义了一个名为变量numberOfLegs并包含三个键-值对字典的文字初始化。的类型的numberOfLegs字典被推断为[String: Int]。创建字典之后，本示例使用标分配来添加String的键bird和Int的值2到词典中。

有关详细信息，Dictionary下标，请参阅访问和修改字典。

注意

斯威夫特的Dictionary类型实现其键值下标作为下标这需要并返回一个可选类型。对于numberOfLegs上述解释，key-value的下标取和返回类型的值Int?，或“可选INT”。的Dictionary类型使用一个可选标类型的事实，不是每一个键将具有的值，并以得到的方式通过分配给删除一个关键值建模nil为密钥值。

标选项

标可以采取任何数量的输入参数，并且这些输入参数可以是任何类型的。标也可以返回任何类型。下标可以使用可变参数的参数，但它们不能在超时使用参数或提供默认参数值。

类或结构能提供尽可能多的标实现，因为它需要，和要使用的适当标将基于所述类型中包含的点的下标使用下标括号内的值或值的推断。多个下标这个定义称为标超载。

虽然这是最常见的下标采取单一的参数，也可以定义多个参数的下标，如果它是适合你的类型。下面的例子中定义了一个Matrix结构，它代表一个二维矩阵Double的值。该Matrix结构的标有两个整型参数：

struct Matrix {
let rows: Int, columns: Int
var grid: [Double]
init(rows: Int, columns: Int) {
self.rows = rows
self.columns = columns
grid = Array(repeating: 0.0, count: rows * columns)
}
func indexIsValid(row: Int, column: Int) -> Bool {
return row >= 0 && row < rows && column >= 0 && column < columns
}
subscript(row: Int, column: Int) -> Double {
get {
assert(indexIsValid(row: row, column: column), Index out of range)
return grid[(row * columns) + column]
}
set {
assert(indexIsValid(row: row, column: column), Index out of range)
grid[(row * columns) + column] = newValue
}
}
}
Matrix提供了采用两个参数称为初始化rows和columns，并创建一个数组，它是足够大的存储rows * columns类型的值Double。矩阵中的每个位置被给定的初始值0.0。为了实现这一目标，阵列的大小，并以初始细胞值0.0，被传递到创建和初始化正确大小的一个新的数组的数组初始化。这个初始化更详细描述创建一个带有默认值的Array。

您可以构建一个新的Matrix通过传递适当的行和列数其初始化实例：

var matrix = Matrix(rows: 2, columns: 2)
上面的例子创建一个新的Matrix两行两列实例。在grid此数组Matrix实例实际上是矩阵的扁平版本，从左上角到右下角写着：

图片：../Art/subscriptMatrix01_2x.png
在矩阵的值可以通过传递行和列值到下标，用逗号分隔进行设置：

matrix[0, 1] = 1.5
matrix[1, 0] = 3.2
这两个语句调用标的setter设置的一个值1.5的矩阵（其中的右上方位置row是0与column是1），并3.2在左下位置（其中，row是1和column是0）：

图片：../Art/subscriptMatrix02_2x.png
该Matrix标的getter和setter都包含一个断言，以检查标的row和column值是有效的。为了帮助这些说法，Matrix包括被称为一个方便的方法indexIsValid(row:column:)，检查是否有要求row和column有基体的边界内：

func indexIsValidForRow(row: Int, column: Int) -> Bool {
return row >= 0 && row < rows && column >= 0 && column < columns
}
如果您尝试访问该标是矩阵范围之外的断言被触发：

let someValue = matrix[2, 2]
// this triggers an assert, because [2, 2] is outside of the matrix bounds
